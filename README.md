# ☕ Java Programming Practice

This repository contains a collection of **Java programs covering fundamental programming concepts**.
It is designed for beginners who want to learn Java step-by-step through simple examples and practice problems.

The programs demonstrate core programming concepts such as **variables, input handling, conditional statements, loops, and pattern printing**.

---

# 📚 Topics Covered

The repository is organized into folders based on Java concepts.

### 1️⃣ Introduction to Java

Basic program structure and first Java program.

* First Java program
* Compilation and execution

### 2️⃣ Variables

Understanding how variables work in Java.

* Variable declaration
* Data types
* Storing values in variables

### 3️⃣ User Input

Programs that take input from the user.

Examples include:

* Reading numbers
* Calculating sum of numbers
* Using `Scanner` class

### 4️⃣ Conditional Statements

Programs demonstrating decision making in Java.

Concepts covered:

* `if` statement
* `if-else`
* `else-if`
* `switch` statement

### 5️⃣ Loops

Examples demonstrating different looping structures.

Programs using:

* `for` loop
* `while` loop
* `do-while` loop
* Sum of natural numbers

### 6️⃣ Pattern Problems

A large collection of pattern printing problems to practice loops and logic.

Examples include:

* Half Pyramid
* Inverted Pyramid
* Number Pyramid
* Floyd’s Triangle
* Butterfly Pattern
* Diamond Pattern
* Hollow Rectangle
* Palindromic Pyramid
* Boolean Triangle

Pattern problems help strengthen **logic building and nested loop understanding**.

# ⚙️ Requirements

To run these programs you need:

* **Java JDK 8 or higher**
* Any IDE such as:

  * IntelliJ IDEA
  * Eclipse
  * VS Code
  * or simple terminal

Check Java installation:

```
java -version
```

---

# ▶️ How to Run the Programs

### 1. Compile the Java file

```
javac FileName.java
```

Example:

```
javac First.java
```

### 2. Run the program

```
java FileName
```

Example:

```
java First
```
